Tags and their applications

The <html> tag
The <html> tag defines the root of an HTML document. All the HTML content is contained within the opening and closing tags of the <html> tag.
The <title> tag
The <title> tag is used within the opening and closing tags of the <head> tag. This is used to define the title of the webpage that is shown in the browser's title bar
The <p> tag
The <p> tag is used to define a paragraph in a webpage. This adds a single blank line before and after each <p> element. This is a container tag which means that its is a tag with an opening and closing tag.
The <br> tag
This is an empty tag. This means that it is a tag with only one opening and no closing tag. The <br> tag is used to break a line and start from a new line.
The <img> tag
The <img> tag is an empty tag. This is used to add image to a webpage. The <img> tag standing alone cannot add image, the src attribute is essential with the <br> tag.
